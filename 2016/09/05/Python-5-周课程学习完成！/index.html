Python 5 周课程学习完成！

发表于 2016-09-05   |  

最后一个大作业，花了一整天时间做完了。作业介绍里，光是步骤就有14个。我竟然完成了。从早上的畏难情绪、上午因为恐惧逃避心理、边看视频边打瞌睡、下午克服畏难心理振作起来，认真按照步骤说明一点点做，不懂的就回看视频。到了晚上八点三十九分，终于把Pong游戏给整出来了！别说，玩自己写的游戏真的很High！

但是，不管怎么样，结果总算是完成了！实在太开心！虽然现在还没有提交作业，但已经迫不及待地想要广播出来！

我！完！成！了！
