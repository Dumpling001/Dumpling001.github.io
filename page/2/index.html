引言

大多数人认为语言是一项文化创造，达尔文认为是掌握一项技艺的本能倾向，詹姆斯认为是本能，乔姆斯基建立了普遍语法理论。

语言的神奇：可以传递想法，跨越时空和血缘。
语言在人类进化史上有重要作用，与日常生活密切相关。
本书探讨人学习、表达和理解语言的本能。

文化创造论：语言将人与动物区分开。语言与思想联系紧密。
语言本能论：语言能力是人脑中的一个构件，一种生物属性。语言的运用是本能，但语言系统本身丰富而精美。
普遍语法：1）有限的词造出无限的句子；2）人能本能地领悟从未见过的句式。

提出问题：为什么说语言是人类的一种生物属性或一种本能？

01有人类存在的地方，就有语言存在

语言的普遍性（高地人自有语言）
文明程度与语言复杂精细程度无关（黑人语言更高明）
克里奥语的发展历史充分说明了大脑有先天语法机制，成功的语言学习有关键期，
儿童天生就具备语法逻辑，婴儿的牙牙学语证明人类头脑中的确存在语法规则的先天蓝本。3岁是儿童的语法爆炸期。语言的获得是自己的创造而不是模仿。

布洛卡氏失语症，证明了大脑是语言的生物学基础。有人智力正常，但语言能力低下；有的智力低下，但语言能力良好。

02心语

许多文章认为语言决定思维。
空的汽油桶事例只能证明人被眼睛蒙骗而不是语言的问题；
“白色向下运动”采用了生硬的逐字翻译法；
颜色与语言无关，只与视网膜上的视锥细胞相关；
虚拟语气结构的例子只因为翻译拙劣。

聋哑人、婴儿、猴子、创作者、物理学家都有用非语言思维的例子。
认知科学家发明了心理旋转实验。
符号表征系统——图灵机器，即心理表征理论，是认知科学的基础。
语言不能用作心智运算的介质。
思维表征和语言有许多不相合。
人是依靠思维语言思考的。思维语言是一种普遍的心语。
掌握一门语言就是懂得如何将心语翻译一串串文字，或者将一串串文字翻译成心语。

语言机制

语言的奥秘：
1）符号的任意性原理，即声音与意义的匹配完全是约定俗成的结果。
2）人类在使用一种规则在字序和思想之间进行转译。这个规则即生成语法。

语言的工作机制：
每个人的大脑中都包含了一部心理词典，内容是所有词语及其代表的概念；还有一套心理语法，即遣词造句的规则，用来传达概念间的逻辑关系。

语法是离散组合系统，有两个特性：
1）有限域的无限应用；
2）与认知（或者说意义）无关。

字串机是最简单的离散组合系统，可通过语法组合文字，但不能代替语言机制，后者有字串机所没有的三个特性：
1）出现概率小的字序连接也可以符合语法，组成句子；但只根据跃迁概率串联起的字串不一定符合语法；
2）语言前后单词有长距离依存关系（如either…or，if…then），还可以相互嵌套，产生无限长的句子，字串机无法处理；
3）句子是模块化设计，短语之间能够随意耦合，因此语法有自洽性。短语结构使得单词之间的树状关联反映出心语中各个概念之间的关系，而字串机只能顺序逐字拼接。

隐性短语结构决定了句子的含义：
语言不同于思想，因为同一个句子可能有几种的解释。这些解释均源于句子中单词所构成的不同树形图。

乔姆斯基的语言机制理论：
短语结构是语言的构造材料。
人们想了解心智，却不愿意读乔姆斯基的经典著作。

名词，是可以被识别、计算和测量的概念；
动词，表示某个事件或者状态（涉及一些相互影响的参与者）的一类概念。

通过词义构建短语意义的原则：
1）短语的意义取决于中心语的意义；
2）短语的“扮演角色”指出了一组关系中的参与对象；
3）构成短语的第三个成分是修饰语，可能一个也可能多个；
4）短语的第四个成分是指示语，即行为发出者。

名词短语和动词短语的构成都一样，内部排列顺序也一样，中心语位于扮演角色之前，修饰语位于它们的右侧，主语位于它们的左侧（指英语）。

短语的超级规则：
1）一个短语由一个可有可无的主语，加上一个扮演角色，再加上任意数目的修饰语构成；
2）一个扮演角色由一个中心语，加上任意数目的扮演角色构成。

乔姆斯基认为，这种与次序无关的超级规则普遍存在，与生俱来，儿童学习语言，只需要学习特定语言的参数值：是像英语一样，中心语在前，还是像日语一样，中心语在后。

这可以解释儿童的语法爆炸期：他们只需要掌握几个心智按钮就可以了，而不需要掌握几十、几百种规则。

JAVA

5）安全性
Java被设计能够防范以下问题：

  • 运行时堆栈溢出；
  • 自己的处理空间之外破坏内存；
  • 未经授权读写文件；有数字签名类；
  • 在程序运行时加以控制并制止恶意破坏，与ActiveX比起来更安全。

6）体系结构中立
Java编译器通过生成与特定的计算机体系结构无关的字节码指令，来生成一个体系结构中立的目标文件格式。
可以将使用最频繁的字节码序列翻译成机器码（即时编译）。

7）可移植性
Java的数据类型的大小和算法都有明确规定，字符串用标准的Unicode存储；
类库定义了可移植的接口；
用户界面工具箱虽然不好看但很实用，并且现在已经完全重写，更稳定。

8）解释型
移植了解释器的机器上都可以执行Java字节码，现在使用即时编译器将字节码翻译成机器码。

9）高性能
即适用性更强。字节码可以在运行时刻快速翻译成应用程序的特定CPU的机器码。
即时编译器可以监控经常执行哪些代码并优化代码以提高速度，或者何时内嵌何时消除内嵌，甚至撤销优化。

10）多线程
只要操作系统支持，Java中的线程就可以利用多个处理器；
它把多线程的实现交给了底层的操作系统或线程库来完成；
多线程编译的简单性让Java成为服务端开发语言。

11）动态性
库可以自由添加新方法和实例变量；
可以往正在运行的程序中添加代码；
可以在程序运行时获得对象的结构和行为等类型信息。

JAVA

1）简单性

简单性是与C++对比得出的特性。它没有C++的头文件、指针运算、结构、联合、操作符重载、虚基类等。
Java还能支持开发能够在小型机器上独立运行的软件。

2）面向对象
面向对象特性与C++相当，不同点在于多继承。Java使用的是简单的接口概念和元类模型。

3）网络技能
能够通过URL很方便地打开访问网络上的对象。

4）健壮性
与C++的不同在于，Java采用的指针模型可以消除重写内存和损坏数据的可能，不会出现内存分配错误，也不必防范内存泄漏。
Java编译器能够检测其他语言在运行时才能发现的问题。
