class可以制作很多个，同时调用很多次，相互之间不会干扰。但导入的模块，在整个程序中只有一个。

instantiate
用具体例证说明
例示

当你用具体例证演示一个class时，你得到一个对象。

通过调用class，你得到一个由类创造的实例。

  • dict style
    mystuff[‘apples’]

  • module style
    mystuff.apples()
    print mystuff.tangerine

  • class style
    thing = MyStuff()
    thing.apples()
    print thing.tangerine

如果不使用self.cheese,代码不知道你是指实例的属性，还是指一个局部变量叫cheese。用了self.cheese=‘Frank’代码就清楚了这是指实例的属性。

class X(Y)
“Make a class named X that is-a Y.”
class X(object): def init(self, J)
“class X has-a init that takes self and J parameters.”
class X(object): def M(self, J)
“class X has-a function named M that takes self and J parameters.”
foo = X()
“Set foo to an instance of class X.”
foo.M(J)
“From foo get the M function, and call it with parameters self, J.”
foo.K = Q
“From foo get the K attribute and set it to Q.”
