Python Crawler

解决了一个大问题：jQuery过滤条件过期

问题记录：
The reason is that, an url function is out of date, like this:

return doc(‘div[class=”wx-rb bg-blue wx-rb_v1 _item”]’).attr(‘href’)

and there is no problem after updating it by:

return doc(‘a[uigs=”account_name_0”]’).attr(‘href’)

https://github.com/ipython/ipython/issues/10379

自己报错信息，一步步看程序调用发现的。先是发现把一个url屏蔽后，程序可以跑通，只是什么都没抓到。所以推测问题就出在网址解析上。接着分析了代码，猜测是过滤条件更新，通过关键词pq().attr()查找发现是pyquery的问题，在pyquery的帮助文档关于绝对链接部分找到对应语法，然后查找了W3C School上jQuery的语法，参考以上信息和自己的猜测（直觉像正则表达式一样），改了下过滤条件，整个代码就跑通了。
