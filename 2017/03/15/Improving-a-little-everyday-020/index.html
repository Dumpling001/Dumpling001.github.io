问题：
from ex47.game import Room
ImportError: cannot import name Room

解决：
把setup.py文件里的 ‘name’: ‘project_name’改成ex47就OK了，game.py 是放在跟tests文件夹平行的文件夹ex47下面。

stuff = raw_input(‘> ‘)
words = stuff.split()

元组就是一个不能更改的列表。把数据放到（）里，数据之间用逗号隔开，就创建了一个元组。

Imort错误通常由4种原因造成：
1）有模块的目录下没有init.py文件；
2）你所在目录不对；
3）因为拼写错误，你在导入错误的模组；
4）PYTHONPATH没有设置，因此你不能从当前目录中下载模组。
