Trouble Shooting

昨晚花了2个小时解决了2个问题：

1）java 安装库源文件和文档，

是《JAVA核心技术（卷1）》上指导步骤，
它要求先找到jdk所在目录，在电脑上所有可能地方翻，没找到；

搜索了以下关键词：

  • jdk安装后需要重启电脑吗(我真的重启了，没用)
  • mac jdk安装路径 （找到：——mac新手，怎么配置jdk路径segfaulment）
  • mac 找不到jdk——（找到：mac安装jdk1.7找不到路径?）
  • mac jdk1.8.0——（找到oracle官网文档：JDK 8 Installation for OS X，没有提及库文件的事）

解决办法：忽略

2）启动shell报错“alias: -= invalid option”

搜索关键词：

  • mac 搜索 文件内容
  • 查看 ./bash_profile
  • 查看 bash_profile
  • bash_profile

  • open bash_profile——（找到：How do I edit $PATH (.bash_profile) on OSX? stackoverflow，在那里搜“alias: invalid option”）

  • alias在哪里

  • shell启动时会用到哪些文件——（找到：bash启动脚本）

然后在电脑下ls -a
发现有好多跟bash相关的文件，其中有个叫.aliases, nano aliases打开,看到确实有个 alias cd -= “cd -”的表达式，然后把这条表达式删掉了，再$ source .aliases,没有再报错，关掉terminal再开，也没有报错。说明问题解决了。

中间改了好多次.bash_profile幸好没有出大错。

通过之前在Codecademy和Learnpythonthehardway上对Command line的学习，加上这次的折腾，对shell算是彻底了解了。加上这几天的学习对python的运行方式也加深了理解。
