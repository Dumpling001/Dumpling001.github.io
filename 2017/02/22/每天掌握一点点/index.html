SQL:

选取某一列：
SELECT 列名称 FROM 表名称

选取所有列：
SELECT * FROM 表名称

选取某一列（去除重复的记录）：
SELECT DISTINCT 列名称 FROM 表名称

有条件地选取某列，条件用 “WHERE 列 运算符 值”表示
SELECT 列名称 FROM 表名称 WHERE 列 运算符 值

仅选取符合条件的记录：
SELECT * FROM 表名称 WHERE 列 运算符 值

linux:

cd ../path1 切换到上一层目录路径
cd ./path2 切换到当前目录路径
cd ／path3 切换到路径

ls 列出当前目录下的文件信息
ls -a 列出当前目录下全部文件，包括隐藏文件
ls -l 列出当前目录下文件的全部属性

Python:

优点：

  • 可进行网站建设工作；
  • 可编写简短而粗糙的小程序；
  • 有一个巨大而活跃的科学计算社区；
  • 能够轻松集成C、C++、Fortran；
  • 适用于研究和原型构建，同时适用于生产系统的构建。

不足：

  • 是一种解释型语言，大部分比编译型语言（Java和C++）运行慢得多；
  • 有一个全局解释器锁，防止解释器同时执行多条指令，所以不适用于高并发多线程场景；
