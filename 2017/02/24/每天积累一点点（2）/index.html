SQL:

SELECT 列名称1, 列名称2 FROM 表名称 ORDER BY 列名称1或列名称2
根据制定列对结果排序

默认为升序，如果要降序排列，加上DESC

SELECT 列名称1, 列名称2 FROM 表名称 ORDER BY 列名称1 DESC

Unix:

在OS系统上测试的。

cd foldername
可以用tab健联想

pwd
显示当前所在目录路径

rm filename
删除文件，不可恢复

Python:

  • NumPy：科学计算的基础包
  • Pandas：提供大量数据结构和函数，方便快速处理结构化数据
  • matplotlib：用于绘制数据图表
